# Static Site Skeleton

A project skeleton (using [Gatsby.js](https://www.gatsbyjs.org/)) for building static PWAs.

## Up And Running

```Bash
git clone git@github.com:reecelucas/gatsby-site-skeleton.git
cd gatsby-site-skeleton
yarn
yarn develop
```
